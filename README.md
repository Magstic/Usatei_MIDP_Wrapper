# Usatei MIDP Wrapper

A MIDP wrapper for the DoJa platform game *その兎、暴走。*.

It includes wrappers such as D4D → M3G and MLD → MIDI.

## How to Use?

1. Put the DoJa version JAR into `Original/`;
2. Run `ant`;
3. The MIDP version of Usatei will be output to `dist/` :)

You need to configure `Apache Ant` and the `J2ME SDK` first, and place `antenna-bin-1.2.1-beta.jar` under `lib` :)

## What Devices Is It Suitable For?

Not real devices, because it is driven entirely by wrappers :(

You can run this game on any J2ME emulator with full M3G support!
